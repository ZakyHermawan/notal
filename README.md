# notal
compiler for pascal-like language 

# Dependencies
1. flex 2.6.4
2. bison (GNU Bison) 3.5.1
3. llvm-12
4. g++ 9.3.0
5. GNU Make 4.2.1

# Contibution guide
1. Add or fix token in toy.l
2. Add or fix grammar and semantic actions in toy.y
3. Add or fix code generation in .cpp files

# Resources

2. Flex Documentations
